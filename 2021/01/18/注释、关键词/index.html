注释、关键词

发表于 2021-01-18 | 分类于 Java基础
字数统计: 751 | 阅读时长 ≈ 2

1.注释

帮助开发者更好地读懂代码，用通俗易懂的语言对代码进行解释说明，方便自己和他人阅读。

注释可以是编程思路，也可以是功能描述或者程序的作用，总之就是对代码的进一步阐述。

注释是专门给程序员看的，计算机直接跳过。

Java 中有几种注释？

1、单行注释

2、多行注释

3、文档注释

IDEA/Eclipse 可以自动生成文档注释，根据 JDK 的 javadoc 工具来自动生成，直接显示到代码中。

javadoc 注释

标签 描述
@author 标识作者
@param 标识方法的参数
@renturn 标识方法的返回值
@version 标识版本
@since 标识从哪个版本开始引入
@see 标识参考的内容（相关的类有哪些）

2.关键字

Java 语言预先定义好的，具有特殊意义的标识符（单词），是程序的核心组成。

Java 程序：关键字 + JDK类库 + 逻辑代码

关键字构建程序的主体框架

JDK 类库提供某些功能的调用

逻辑代码实现程序的业务逻辑的

IDEA 中橙色字体显示的都是关键字 public、static、int、void…

关键字 含义
case 与switch组合使用，进行流程控制
catch 捕获异常
char 基本数据类型
class 声明一个类
continue 跳出循环
default 与switch组合使用，表示默认值
do 与while组合使用，表示循环
double 基本数据类型
else 与if组合使用，进行流程控制
enum 枚举类型，Java 中的一种特殊的类
extends 实现类的继承
final 修饰常量
finally 与try、catch搭配使用，处理异常
float 基本数据类型
for 循环
if 流程控制
implements 实现接口
import 引入某个类
int 基本数据类型
interface 接口
instanceof 判断对象的类型
long 基本数据类型
native 修饰本地方法
new 创建实例对象
package 表示包，用来组织 Java 类的
private 访问权限修饰符，表示私有
protected 访问权限修饰符，表示保护
public 访问权限修饰符，表示公开
return 表示方法的返回值
short 基本数据类型，短整型
static 静态修饰符
super 表示父类的成员
switch 流程控制，和case组合使用
synchronized 线程同步
this 表示调用当前实例对象的成员
throw 抛出异常，与 try、catch、finally 组合使用
throws 标识方法中可能抛出的异常
try 异常处理，与 catch、throw、finally 组合使用
void 表示当前方法没有返回值
volatile 确保主内存对所有线程可见
while 循环结构
